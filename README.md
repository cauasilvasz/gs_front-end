Projeto Global Solution Front End Design Web Development

Este projeto foi desenvolvido como parte da disciplina de Front End Design Web Development. A proposta consiste na criacao de uma plataforma web inspirada no LinkedIn, com foco no futuro do trabalho. A aplicacao apresenta uma lista de profissionais ficticios exibidos em cards e permite visualizar informacoes completas de cada perfil por meio de um slideshow. O objetivo e conectar pessoas, competencias e propositos utilizando tecnologia.

A aplicacao foi desenvolvida com HTML CSS e JavaScript e inclui recursos como animacoes flexbox transicoes quiz e interacao com dados armazenados em arrays.

Resumo do projeto

Criacao de uma plataforma web simulando uma rede profissional com listagem de perfis cards slideshow informacoes pessoais habilidades experiencias soft skills hobbies e quiz sobre o mercado de trabalho.
